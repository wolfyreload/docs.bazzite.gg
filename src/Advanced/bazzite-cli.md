## List of CLI Tools

```
ujust bazzite-cli
```
Bazzite's opt-in command line experience. `bazzite-cli` comes with some fantastic command line tools such as:
- [**bat**](https://github.com/sharkdp/bat) - A `cat` clone with syntax highlighting and Git integration
- [**bash-preexec**](https://github.com/rcaloras/bash-preexec) - Preexec and precmd functions for Bash (required by atuin)
- [**bbrew**](https://github.com/valkyrie00/bbrew) - Homebrew wrapper for better Bazzite integration
- [**atuin**](https://github.com/atuinsh/atuin) for shell history
- [**chezmoi**](https://www.chezmoi.io/) - Manage your dotfiles across multiple diverse machines, securely
- [**direnv**](https://direnv.net/) to load and unload environment variables depending on the current directory.
- [**dysk**](https://github.com/Canop/dysk) - A Linux utility to get information on filesystems, like df but better
- [**eza**](https://github.com/eza-community/eza) as a replacement `ls`
- [**fd**](https://github.com/sharkdp/fd) for `find`
- [**gh**](https://cli.github.com/) - GitHub CLI for managing GitHub from the command line
- [**glab**](https://gitlab.com/gitlab-org/cli) - GitLab CLI for managing GitLab from the command line
- [**ripgrep**](https://github.com/BurntSushi/ripgrep) for search
- [**shellcheck**](https://www.shellcheck.net/) - Shell script static analysis tool
- [**starship**](https://starship.rs/) - The minimal, blazing-fast, and infinitely customizable prompt for any shell
- [**stress-ng**](https://github.com/ColinIanKing/stress-ng) - Stress testing tool for Linux
- [**tealdeer**](https://github.com/dbrgn/tealdeer) for `tldr`
- [**television**](https://github.com/alexpasmantier/television) - A blazingly fast general purpose fuzzy finder TUI
- [**trash-cli**](https://github.com/andreafrancia/trash-cli) - Command line interface to the freedesktop.org trashcan
- [**ugrep**](https://github.com/Genivia/ugrep) for grep
- [**yq**](https://github.com/mikefarah/yq) - for yaml, json, and xml processing
- [**zoxide**](https://github.com/ajeetdsouza/zoxide) as `cd`

The community may add new tools over time, re-running `ujust bazzite-cli` will pull in the new tools.
